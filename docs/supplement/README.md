# docs/supplement

Extended analysis and discussion that supports, but is not part of, the
algorithm specifications in `docs/algorithm/`.

## Contents

- **`phase_space_crystal_lattice_supplement.md`** — Redrafted and restructured
  version of David Cyganski's project memo *Extended Fokker–Planck Eq. and
  the QLE V2*, focused on the phase-space crystal-lattice particle model and
  its stochastic microdynamics. Embeds key figures from his Sozi slide deck.
  Applies one sign correction to the V2 simplified-form discrete update; see
  §6.3 of the document.
- **`figures/`** — Figures cited by the supplement, reproduced from
  Cyganski's *Wigner Collisions Diagram* Sozi presentation.
